# Basic questions
### Q: how do I use this?
A: depends on what you want to do with it, I plan to add 'gif characters' but right now the [README](https://github.com/melodiiCam2B/funkingif/blob/main/README.md) has everything you need to use it.

### Q: can I use this freely in my basegame/ ""v-slice"" mods?
A: yes if you use it externally and with credit, no if you duplicate the code into your mod and ship it that way. 
why? because it could cause module errors because of duplicate files.

### Q: can I fork it?
A: absolutely, it's free to use as long as credit is given to me and the original creators [MAJigsaw77](https://github.com/MAJigsaw77/flxgif) and [yanrishatum](https://github.com/yanrishatum/yagp) since their code is the base of mine.
